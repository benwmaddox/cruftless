Getting Started with Cruftless CSS

A minimal example showing how to use the reset and design tokens.

Modern CSS Made Simple

Import the reset, use the tokens, write clean CSS.

Quick Start

1. Import the reset

<link rel="stylesheet" href="reset.css">

2. Use design tokens

.my-component { padding-inline: var(--space-4); padding-block: var(--space-3); background: var(--color-primary); box-shadow: var(--shadow-sm); }

3. Modern layout with flexbox

✅ Flexbox

Use flex for one-dimensional layouts with gap for spacing.

✅ Grid

Use grid for two-dimensional layouts with auto-fit and minmax.

✅ Logical Properties

Use margin-inline, padding-block for internationalization-ready styles.

What you get out of the box:

  • Border-box everywhere - predictable sizing
  • System fonts - fast loading, native look
  • Responsive media - images scale automatically
  • Reset margins/padding - consistent starting point
  • Design tokens - semantic spacing, colors, z-index
  • Container queries ready - elements adapt to their container
