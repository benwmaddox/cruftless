Modern CSS Layout Demo

This page demonstrates modern CSS features using only the Cruftless CSS subset:

  • Flexbox & Grid - for predictable layouts
  • Container Queries - components adapt to their container
  • Logical Properties - margin-inline, padding-block
  • Semantic Z-index - layered with design tokens
  • Design Tokens - consistent spacing, colors, shadows

Flexbox Layout

The header uses flexbox for alignment. The main container uses flexbox with gap for spacing.

CSS Grid

This card grid uses CSS Grid with auto-fit and minmax for responsive behavior without media queries.

Container Queries

Resize the browser to see how the sidebar and cards adapt based on their container size, not viewport.

Design Tokens

All spacing uses semantic tokens (--space-4), z-index uses layers (--z-overlay), colors use variables.

Modal Example

This modal is layered using semantic z-index tokens (--z-modal) for predictable layering.
